# Sanskriti-Flow Frontend

This is the frontend dashboard for Sanskriti-Flow, built with Next.js 15 and Tailwind CSS.

## Getting Started

### Install Dependencies
```bash
npm install
```

### Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

### Build for Production
```bash
npm run build
npm start
```

## Features

- Video URL submission with localization options
- Language selection (10+ Indian languages)
- Interactive quiz generation toggle
- Vision-sync overlay toggle
- Job status tracking
- Responsive design
- Dark mode support

## Environment Variables

Create a `.env.local` file:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Tech Stack

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Axios for API calls
- Framer Motion for animations
